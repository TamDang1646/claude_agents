# Skill: Documentation Sync

Use this skill when code, commands, configuration, public behavior, architecture, or operational workflows change.

## Check

- README
- CLAUDE.md
- docs directory
- API docs
- migration notes
- runbooks
- examples
- command references

## Rules

- Update docs only where they are the source of truth.
- Keep documentation factual and aligned with code.
- Do not add broad tutorial content for narrow changes.
- Mention docs skipped when no source-of-truth doc exists.

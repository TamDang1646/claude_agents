Create a handoff summary for the current repository work.

Optional focus: $ARGUMENTS

Follow `.claude/team/contracts.md`, `.claude/team/workflows.md`, and `.claude/team/communication.md`.

Include:

- Goal
- Current state
- Decisions made
- Files changed or inspected
- Commands run and results
- Known blockers
- Next recommended action
- Handoff envelope fields when another agent or future session will continue

Keep it concise and factual so another Claude session can continue without re-discovering context.

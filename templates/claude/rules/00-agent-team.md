# Agent Team Rule

This repository uses Claude Code team configuration.

- Start with `CLAUDE.md`.
- Use subagents for focused delegated work that reports back to the coordinator.
- Use experimental agent teams only when teammates need to communicate directly, debate, or coordinate independent parallel work.
- Use `.claude/team/communication.md` to decide whether communication should be coordinator-mediated or teammate-to-teammate.
- Use `.claude/team/contracts.md` for output shape.
- Use project skills in `.claude/skills/` for reusable workflows.

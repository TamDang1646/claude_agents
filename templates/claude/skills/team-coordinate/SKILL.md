# Team Coordinate Skill

Use this skill to choose between direct work, subagents, and experimental agent teams.

Read:

- @../../team/manifest.md
- @../../team/communication.md
- @../../team/agent-teams.md
- @../../team/contracts.md
- @../../comms/README.md

## Decision

- Direct work: simple one-step task.
- Subagent: focused work where only the result needs to return.
- Agent team: complex parallel work where teammates must communicate directly, debate, or coordinate independent file ownership.

## Output

- Chosen mode
- Agents or teammates used
- Task split
- Communication plan
- Local artifact plan
- Validation/review gate
- Final synthesis
